Project 1 Details

TakboLipa is a concept prototype developed as part of a group project for Algocom (Algorithm and Complexity), focusing on an algorithm-powered traffic management system for Lipa City.

While the team explored route optimization logic and traffic data processing, I designed the UI/UX—creating a mobile-first interface that visualizes real-time traffic monitoring, optimized route suggestions, and integration of local factors like school schedules and events.

Note: This is a non-functional UI mockup only—it does not include actual algorithms, real-time data, or responsive behavior. The prototype serves as a visual representation of the proposed solution.

Project 2 Details

RecipeNest is an AI-powered web app prototype designed for our Software Engineering subject. It helps users generate custom recipes based on available ingredients, manage kitchen inventory, track food expiration dates, and learn cooking techniques through interactive guides.

As part of a team, we contributed to frontend development, UI design, and features needed. The prototype demonstrates core functionalities like recipe generation and inventory management, though it lacks full AI integration or backend scalability.

Note: This is a proof-of-concept prototype only—not a deployed product. Data is simulated, and some features are simplified for academic purposes.

Project 3 Details
Student Enrollment System is an individual project developed for InfoMan (Information Management). This system allows administrators to manage student records, course offerings, and class lists in one centralized platform. Key features include enrolling/dropping students from courses, editing student profiles, and tracking academic schedules.

Designed with efficiency in mind, the system demonstrates CRUD (Create, Read, Update, Delete) operations for student data management. The interface prioritizes navigation for common tasks like course registration and class roster management.

Note: This is a functional prototype demonstrating core enrollment workflows. While it includes basic validation, it doesn't represent a production-ready system with full scalability or advanced security features.
